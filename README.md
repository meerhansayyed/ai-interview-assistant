# 🤖 AI Interview Assistant

An AI-powered mock interview platform with:
- 🎙️ Voice + text answer support
- 🧠 Real-time sentiment & confidence analysis
- 📊 Detailed scorecards with hire recommendations
- 🛡️ Admin dashboard with analytics
- 📄 PDF report export

## Built With
- **Streamlit** — UI framework
- **Groq + Llama 3.3 70B** — Interview AI (free tier)
- **Whisper** — Voice transcription
- **SQLite** — Interview storage
- **ReportLab** — PDF generation
